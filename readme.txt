Installer les choses suivantes:
node.js -> vérifier qu'il fonctionne bien avec la commande "node" ou "npm" (il faut parfois redémarrer);
taper la commande suivante "npm install" -> l'installation des packages nécessaires va se faire;
Installation terminée;

Pour le dév:
taper commande "gulp move";
taper commande "gulp dev";
faire les modifs, elles seront automatiquement pris en compte par gulp qui va s'occuper de régénérer les sources pour notre serveur statique;
l'URL auquel on doit accéder pour vérifier les modifs est : http://localhost:8080

Pour upload:
tous les fichiers nécessaires sont dans le répertoire dist/
